# Memory Strategies

## Long-term Memory

### AGENTS.md Pattern

```markdown
# Project Context
- Key technologies: React, Node.js, PostgreSQL
- Build commands: npm run build, docker-compose up

# Code Style
- TypeScript strict mode enabled
- ESLint + Prettier configuration
```

### Skills Repository

- Store reusable patterns in skills
- Version control for knowledge
- Share across projects

## Short-term Memory

### Conversation Context

- Summarize after 10+ exchanges
- Flag critical information
- Use task tracker for complex work

### Tool Output Caching

- Cache expensive operations
- Reuse file contents
- Avoid redundant API calls

## Optimization

1. **Prioritize**: Critical > Important > Contextual
2. **Summarize**: Replace verbose with concise
3. **Index**: Use consistent terminology
4. **Prune**: Remove stale information
