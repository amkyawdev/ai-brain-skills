## Overview

Strategies for managing persistent and working memory across sessions.

## Memory Types

### Persistent Memory
- AGENTS.md files in repository root
- Skills repository
- External knowledge bases

### Working Memory
- Current conversation context
- Active task state
- Tool outputs

## Strategies

1. **Active Recall**: Explicitly reference stored information
2. **Pattern Recognition**: Identify recurring themes
3. **Structured Storage**: Use consistent formats
4. **Periodic Review**: Update memory based on outcomes
