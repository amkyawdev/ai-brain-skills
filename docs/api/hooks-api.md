# Hooks API

## Pre-Skill Hooks

### validate-context.js
Validates context before loading skill.

### check-dependencies.js
Checks required dependencies.

### log-invocation.js
Logs skill invocation.

## Post-Skill Hooks

### audit-output.js
Audits skill output.

### update-memory.js
Updates persistent memory.

### telemetry.js
Collects usage metrics.

## Error Hooks

### fallback-handler.js
Handles skill failures.

### notify-on-failure.js
Sends failure notifications.
