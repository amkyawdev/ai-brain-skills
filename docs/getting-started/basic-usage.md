# Basic Usage

## Skill Categories

- **Core (00-core)**: Auto-loaded infrastructure skills
- **Domains (01-domains)**: Domain-specific expertise
- **Custom (04-custom)**: Organization-specific skills

## Loading Skills

### Automatic Loading
Skills load automatically based on file types and context.

### Manual Loading
```
Use skill: backend/python/fastapi
```

## Skill Structure

Each skill contains:
- `SKILL.md`: Main documentation
- `references/`: Detailed references
- `scripts/`: Executable scripts
- `templates/`: Code templates
