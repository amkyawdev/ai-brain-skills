# Installation

## Prerequisites

- Node.js 18+
- npm or yarn
- Python 3.10+ (for some scripts)

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/ai-brain-skills.git

# Install dependencies
npm install

# Validate skills
npm run validate
```

## Configuration

Copy `.env.example` to `.env` and configure:

```bash
cp .env.example .env
```
