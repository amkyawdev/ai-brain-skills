# Markdown Guidelines

## Headers

- Use `#` for main title
- `##` for sections
- `###` for subsections

## Code Blocks

Always specify language:

\`\`\`javascript
const x = 1;
\`\`\`

## Links

Use relative links for internal references:

```markdown
[Reference](./references/guide.md)
```

## Lists

Use bullet points for lists:
- Item 1
- Item 2

Use numbers for sequences:
1. First
2. Second
