# Performance Optimization

## Caching

Enable skill caching:

```json
{
  "cacheEnabled": true,
  "cacheTTL": 3600
}
```

## Lazy Loading

Load references only when needed:

```json
{
  "lazyLoadReferences": true
}
```

## Parallel Loading

Load multiple skills in parallel:

```json
{
  "parallelLoading": true
}
```
