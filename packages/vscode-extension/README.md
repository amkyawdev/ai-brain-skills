# AI Brain Skills - VS Code Extension

Install this extension to manage AI Brain Skills directly from VS Code.

## Features

- 🌳 **Skills Tree View** - Browse all skills in sidebar
- 🔍 **Search Skills** - Quick search across all skills
- ➕ **Create Skills** - Create new skills from VS Code
- 📄 **Open Skills** - Quick access to skill documentation

## Installation

1. Open the extension in VS Code
2. Press `F5` to run in development mode
3. Or package and install the `.vsix` file

## Usage

### Commands

- `amkyaw.skillList` - List all skills
- `amkyaw.skillSearch` - Search skills
- `amkyaw.skillCreate` - Create new skill

### Keyboard Shortcuts

Set your own in VS Code settings.
